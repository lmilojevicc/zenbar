Browser Shortcut Conflict

By default, Zenbar tries to use Ctrl+L, Ctrl+T, and Ctrl+Shift+A. However, Chromium browsers (Chrome, Brave, Edge) reserve these for native features like the address bar and tab search.

To use Zenbar, you must manually assign these shortcuts in the browser's settings.

Step 1

Enable History & Bookmarks

Zenbar works best when it can search your existing browsing data.

Step 2

Remote Suggestions

Optionally enable DuckDuckGo for live search suggestions as you type.

Ready?

You're all set!
